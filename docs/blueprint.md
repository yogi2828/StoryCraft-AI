# **App Name**: StoryCraft AI

## Core Features:

- Script Generation: Generate a screenplay scene using AI based on provided parameters such as title, genre, tone, and language.
- Script Management: Allow users to save, edit, and organize their generated scripts within a personal library.
- Scene Editing: Enable detailed editing of individual scenes including location, time of day, descriptions, and dialogue, with manual and AI-assisted regeneration options.
- PDF Export: Generate downloadable PDF versions of complete scripts or individual scenes for sharing or submission.
- User Authentication: Secure user accounts with login, registration, and profile settings, managed with JWT for secure access.
- Local AI Inference: Leverage a tool to perform local AI processing using Ollama and llama3 for screenplay and scene generation, ensuring user privacy and reduced latency.
- AI Prompting: Continuity-aware tool for dialogue improvement, based on the scenes before it.

## Style Guidelines:

- Primary color: Indigo (#4B0082), drawing inspiration from the dark cinematic theme.
- Background color: Very dark indigo (#1A0033) for an immersive, distraction-free writing environment.
- Accent color: Amber (#FFBF00), analogous to indigo, providing highlights for important elements and call-to-action buttons.
- Body and headline font: 'Literata', a transitional serif font, to give a literary feel appropriate for scriptwriting.
- Use minimalist icons that complement the dark cinematic theme, ensuring they remain unobtrusive to maintain writer focus.
- Implement a writer-focused layout with minimal distractions, using clear sections for script information, scene editing, and available tools.
- Incorporate smooth transitions and auto-save indicators to provide feedback to user actions without disrupting the writing flow.
